# scrum-guide-2020-NL-audio

De Scrum Gids. De definitieve gids voor Scrum: de regels van het spel. Tweeduizendtwintig. Ontwikkeld door de bedenkers van Scrum: Ken Schwaber en Jeff Sutherland.
